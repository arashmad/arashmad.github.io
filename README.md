# Arash Madadi GitHub Pages Profile

Static personal resume/profile page for `https://arashmad.github.io/`.

## Files

- `index.html` — main profile page
- `styles.css` — responsive styling
- `profile-content.md` — full resume content in Markdown
- `.nojekyll` — disables Jekyll processing for plain static hosting

## Deploy

This repository is configured as a GitHub Pages user site. GitHub Pages should serve the site from the `main` branch root.

Expected public URL:

```text
https://arashmad.github.io/
```
